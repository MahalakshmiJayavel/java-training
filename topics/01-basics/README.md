# Basics

## Topics Covered:
1. Syntax
2. Data Types
3. Variables
4. Input and Output

## Summary:
- Java programs start with the `main` method.
- Data types include `int`, `float`, `double`, `char`, `boolean`, etc.
- Variables store data and must be declared with a type.
- Use `System.out.println` for output.
